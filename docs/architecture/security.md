# Segurança

## Diretrizes mínimas
- autenticação baseada em JWT;
- senhas criptografadas com BCrypt;
- segregação por perfis;
- endpoints públicos e privados bem definidos;
- tratamento de erros sem vazar informações sensíveis.

## Observabilidade de segurança
- registrar falhas de autenticação;
- registrar tentativas de acesso negado;
- evitar logs com dados sensíveis.
