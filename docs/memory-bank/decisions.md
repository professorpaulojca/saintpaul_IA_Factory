# Memory Bank — Registro de Decisões

## DEC-001 — Stack padrão
Adotamos Java com Spring Boot no backend, React com TypeScript no frontend e PostgreSQL no banco.
Motivo: stack moderna, didática e com forte aderência ao mercado.

## DEC-002 — Processo em etapas
A geração deve ocorrer em etapas:
1. protótipo
2. especificação
3. backend
4. frontend
5. revisão
Motivo: reduzir improviso e aumentar rastreabilidade.

## DEC-003 — Segurança desde o início
Toda solução deve considerar autenticação, autorização, logs e tratamento de erros.
Motivo: aproximar a geração de uma aplicação real.

## DEC-004 — Diagramas obrigatórios
Toda história deve gerar pelo menos um diagrama de domínio, um fluxo Mermaid e um BPMN.
Motivo: reforçar compreensão antes do código.
