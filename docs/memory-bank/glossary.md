# Memory Bank — Glossário

- **História da aplicação**: descrição narrativa do problema de negócio e do sistema esperado.
- **Memory Bank**: conjunto de arquivos que guarda contexto explícito do projeto.
- **Prompt de ação**: instrução estruturada que aciona uma etapa de geração.
- **Spec**: especificação funcional e técnica da solução.
- **Agente**: papel especializado atribuído à IA.
- **Fluxo**: sequência operacional relevante do negócio.
- **BPMN**: notação de processo usada para representar fluxos de negócio.
- **Flyway**: ferramenta de versionamento de banco de dados.
- **JWT**: token para autenticação stateless.
