# Glossary

- História da aplicação: narrativa principal do domínio e das necessidades.
- Memory Bank: conjunto de arquivos com contexto persistente e explícito.
- Prompt de ação: comando estruturado para uma fase específica da geração.
- Especificação: documento que detalha requisitos, regras, fluxos e modelos.
- BPMN: notação para processos de negócio.
- Mermaid: sintaxe textual para diagramas.
- Camunda: plataforma de automação/processos.
